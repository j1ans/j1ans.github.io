很简单的一题 主要考察header的使用
开头让我们用WLLM浏览器访问 肯定就是检测UA
然后说让我们在本地访问 header有一个头叫X-Forwarded-For 可以轻松绕过

import requests 

import json

header = {

  'user-agent':'WLLM',

  'X-Forwarded-For':'127.0.0.1'

}

url = f"http://node7.anna.nssctf.cn:23001/"

resp = requests.post(url=url,headers=header)

print(resp.text)
